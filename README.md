## Setup development environment

If you don't run the repository in `Gitpod` environment

1. Run `pnpm install`
1. Run `pnpm dlx husky install`
1. Run `pnpm run dev`

## Commitlint

Run `pnpm run cz` to commit instead of `git commit` or using editor UI
